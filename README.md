## Informações Importantes

**O aplicativo ainda não conecta com o Azure**

* A API utilizada pelo app está rodando junto com a Aplicação Web em servidor local
no repositório em [statue-store-web](https://github.com/EspetacularGus/statue-store-web).
* É possível que seja necessário trocar a string de conexão com a API para conseguir rodar o app localmente.
* A biblioteca utilizada para a construção da interface do app pode ser baixada gratuitamente [aqui](http://www.jfoenix.com/).

## Manual do Usuário

### Tela de Login

Aqui o usuário pode realizar o login com qualquer cadastro que já tenha feito, tenha sido ele realizado pelo site ou pelo
próprio app. Caso ele tenha esquecido a senha, pode clicar em "Esqueci minha senha" e será redirecionado para a página do site para 
a troca da senha.

## Tela de Menu

<img src=https://user-images.githubusercontent.com/59635709/72227647-eb1dbc00-357d-11ea-837e-ca65460f65bc.png height=600> <img src=https://user-images.githubusercontent.com/59635709/72227646-ea852580-357d-11ea-96a9-e75a1d9f9c1d.png height=600>

Nessa tela o usuário poderá ver os produtos já personalizados, clicar no **botão "+"** no canto inferior direito para realizar outra personalização, e ao clicar nas três linhas no canto superior esquerdo ativará o menu lateral, onde poderá ver detalhes sobre as compras realizadas na plataforma (a função de compra pelo app será adicionada em futuras atualizações), detalhes sobre a conta, além de encontrar um atalho para o site e um botão _sair_.

## Personalização do Produto

<img src=https://user-images.githubusercontent.com/59635709/72227648-eb1dbc00-357d-11ea-8bd1-2695cfb640af.png height=600> <img src=https://user-images.githubusercontent.com/59635709/72227649-eb1dbc00-357d-11ea-97c9-17c8fd339134.png height=600>

Aqui o usuário poderá escolher se deseja personalizar uma camiseta ou um moletom, além de escolher a cor do produto e o nome que será salvo. Para escolher a estampa desejada, ele deve fornecer acesso ao armazenamento interno do dispositivo, e selecionar a imagem desejada, que será automaticamente redimensionada para que se encaixe no produto.

<img src=https://user-images.githubusercontent.com/59635709/72227637-e953f880-357d-11ea-8825-ab972dca458f.png height=600> <img src=https://user-images.githubusercontent.com/59635709/72227642-ea852580-357d-11ea-9439-006cbc9af409.png height=600>

## Tela de Conclusão

<img src=https://user-images.githubusercontent.com/59635709/72227643-ea852580-357d-11ea-855c-44dae52758a2.png height=600> <img src=https://user-images.githubusercontent.com/59635709/72227644-ea852580-357d-11ea-9a1d-581f74a6016f.png height=600>

Após completar a personalização do produto, uma tela apresentando a camiseta ou moletom personalizado aparecerá, e você terá a opção de voltar ao menu, realizar o download da imagem do produto, ou deletá-lo caso não tenha gostado do resultado. A menos que o produto seja deletado, o produto ficará salvo para futura compra pelo usuário, assim que a funçõa estiver disponível.
